# Forest Corner (Leśny Zakątek) — Cozy Cottage Rental Website

A premium, fully responsive landing page for a cabin rental business located in the scenic regions of Poland. This project focuses on high-end user experience, seamless multilingual communication, robust form security, and automated real-time booking notifications.

🚀 **Live Demo:** [https://incredible-cascaron-af8252.netlify.app](https://incredible-cascaron-af8252.netlify.app)

---

## 🛠️ Features & Functionality

* **Multilanguage Support (PL/EN):** A fully implemented bilingual system allowing seamless switching between Polish and English layouts, expanding the business reach to international tourists.
* **Telegram API Integration:** The dynamic booking form is connected directly to a Telegram bot. When a client submits a booking request, the website instantly fires a real-time notification to the administrator's chat.
* **Custom Frontend Form Validation:** Built-in client-side validation logic that checks user inputs (dates, contact info) before submission, preventing empty or corrupted data from being processed.
* **Custom Touch-Responsive Slider:** An optimized reviews/gallery slider that tracks desktop clicks as well as mobile **swipe gestures** via touch coordinates analysis (`touchstart`/`touchend`), providing native-like app navigation.
* **Responsive Layout:** Perfectly adaptive CSS grid and flexbox structures that ensure a cozy and premium visual aesthetic across desktop, tablet, and mobile displays.

---

## 💻 Tech Stack

* **HTML5:** Semantic architecture, multi-language datasets.
* **CSS3:** Flexbox, Grid layout, custom animations, and mobile media queries.
* **JavaScript (ES6+):** Async/Await for Telegram API interactions, DOM Manipulation, Event Listeners, and Swipe Logic.
* **Telegram Bot API:** Used as a lightweight backend alternative for instantaneous lead notifications.
